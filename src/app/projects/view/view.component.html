Generador de Sudoku

Funciona con 2 etapas:
En la pantalla de carga se genera un sudoku semilla que luego se usa de base para generar otros sudokus.

La otra etapa es validar que el juego se esta resolviendo bien cumpliendo que no se repitan valores por celda,fila y subcelda de 3x3


Otras funciones:

-Barra de progreso que muestra el avance
-Boton ganar que resuelve el sudoku
-Opcion de habilitar deshabilitar ayuda
con los posibles valores validos para una celda

Desarrollado con

Html5
Bootstrap
Javascript
Jquery
